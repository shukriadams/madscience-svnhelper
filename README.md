# madscience-svnhelper

A NodeJS library of svn-related parsers etc.

## Use 

Add to package.json :

    {
        {
            "dependencies": {
                "madscience-svnhelper": "https://github.com/shukriadams/madscience-svnhelper.git#0.0.1"
        }
    }

Import

    const svnhelper = require('madscience-svnhelper')
